# Try It Out

1. Open **[Anibozu](https://anibozu.nasirov.info/)** via browser with ***enabled JavaScript***
2. Enter your **[MyAnimeList](https://myanimelist.net/)** username. ***Your anime list must be public!***
3. Submit the form
4. Wait for a result
